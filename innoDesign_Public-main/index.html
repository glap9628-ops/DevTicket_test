Review Pass

외부 레퍼런스 검토 후 반영한 보완 항목

전체 톤을 갈아엎지 않고, 현재 관리자 콘솔 기준선에 필요한 개선 포인트만 선별 반영했습니다.

Interaction Safety

  • 드롭다운 옵션과 주요 인터랙션 높이를 44px 기준으로 정리
  • 키보드 포커스가 묻히지 않도록 focus ring 강화
  • prefers-reduced-motion 대응 추가

Hierarchy and Density

  • 간격 토큰을 보강해 임의 padding 증가를 줄임
  • 한 영역에 primary action 하나만 두는 기준 추가
  • 카드 중첩과 저대비 회색 텍스트를 반패턴으로 명시

Spacing

  • 카드 기본 패딩 20px
  • 라벨과 입력창 간격 8px
  • 섹션 제목과 설명 간격 10px
  • 요약 카드 메타 텍스트는 6px 이하
