# InnoDesign Public

InnoSaaS 관리자 콘솔의 디자인 기준을 공유하기 위한 공개용 디자인 가이드입니다.

포함 항목:

- `index.html`: 기준 화면 preview
- `design-guide.css`: 공통 스타일
- `design-guide.js`: 상호작용 예시
- `DESIGN_GUIDE_FOR_AI.md`: AI/개발 전달용 규칙 문서
- `DESIGN_FEEDBACK_LOG.md`: 피드백 기록

## Purpose

이 가이드는 문서 설명만으로 디자인을 해석하는 대신, 실제 화면과 컴포넌트 예시를 기준으로 시각적 일관성을 맞추기 위한 목적을 가집니다.

현재 기준:

- 밝은 배경 기반의 안정적인 관리자 콘솔 톤
- 브라우저 기본 UI에 의존하지 않는 커스텀 스타일
- 버튼, 입력, 드롭다운, 상태 패널, 테이블, 모달, 토스트 등 공통 컴포넌트 예시 제공
- AI와 협업 시 복사 가능한 Markdown 가이드 제공

## Usage

브라우저에서 `index.html`을 열면 전체 디자인 가이드를 확인할 수 있습니다.

구성:

- `Design Guide (Preview)`: 실제 시각 기준 화면
- `Design Guide (Markdown for AI)`: 개발자/AI 전달용 규칙 문서
