## 2차 피드백 반영

- 네이티브 `select` 대신 커스텀 dropdown 컴포넌트로 전환
- 상태 패널 아이콘과 텍스트의 수직 정렬 보정
- 버튼과 배지 간격 확대
- 버튼 내 텍스트 잘림 방지를 위한 `white-space`, `line-height`, `flex-shrink` 보정
- 카드와 입력 컴포넌트 패딩, 행간, gap 추가 확대

## 3차 피드백 반영

- `[hidden]` 강제 처리로 dropdown 메뉴가 기본 접힘 상태를 유지하도록 수정
- dropdown 메뉴는 클릭 시에만 열리고, 외부 클릭 또는 `Escape`로 닫히도록 보강
- 제목과 버튼, 상태카드와 텍스트, 버튼과 배지 사이의 간격을 추가 확대
- Markdown 지침을 실제 현재 파일 구조와 클래스 기준으로 상세화
